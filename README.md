# Control System Response Tools

A Python toolkit for analyzing basic control system responses, including first-order and second-order step behavior.

## Features
- Computes first-order step response
- Computes second-order step response
- Organized function-based structure
- Includes sample control parameter data
- Separate parameter viewer script

## Technologies Used
- Python
- CSV file handling
- Control systems concepts
- Numerical calculations

## How to Run
Run the script in Python:

python control_response.py

To view the sample parameter data:

python parameter_viewer.py
